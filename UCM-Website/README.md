# United by Christ Ministries Website

Official website for United by Christ Ministries - a 501(c)(3) nonprofit dedicated to restoring lives through faith, housing, and comprehensive support programs.

## About UCM

Founded in 2018 by Aureliano "Audie" Reyes, UCM provides comprehensive 1, 2, and 3 step programs that address mental, physical, financial, and spiritual needs for those experiencing homelessness, addiction, and life challenges.
