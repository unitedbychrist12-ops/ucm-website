# UCM 501(c)(3) COMPLIANCE GUIDE FOR R SAVE PARTNERSHIP
*Protecting UCM's Tax-Exempt Status While Maximizing Fundraising*

## CRITICAL COMPLIANCE FINDINGS

### üö® **MAJOR RED FLAGS TO AVOID**

**1. PERCENTAGE-BASED FUNDRAISING COMPENSATION**
- **IRS Rule:** "It is NOT appropriate for a nonprofit to compensate a fundraising professional based on a percentage of the money raised"
- **Risk:** Could jeopardize UCM's 501(c)(3) status
- **Solution:** Use salary-based compensation for UCM fundraising activities

**2. PRIVATE BENEFIT VIOLATIONS**
- **IRS Rule:** No part of net earnings can benefit private individuals
- **Risk:** R Save profits benefiting you personally while using UCM name
- **Solution:** Clear separation between UCM and R Save activities

**3. SUBSTANTIAL NON-EXEMPT PURPOSE**
- **IRS Rule:** Must operate "exclusively" for charitable purposes
- **Risk:** Commercial activities becoming primary focus
- **Solution:** Keep UCM's charitable mission as primary purpose
