## Deployment Instructions

### Option 1: Netlify (Recommended)
1. **Create Netlify Account** at netlify.com
2. **Connect to Git Repository:**
   - Upload files to GitHub repository
   - Connect Netlify to GitHub
   - Select repository and deploy

3. **Custom Domain Setup:**
   - Add RSave.net as custom domain in Netlify
   - Update DNS records:
     ```
     Type: CNAME
     Name: www
     Value: [your-netlify-subdomain].netlify.app
     
     Type: A
     Name: @
     Value: 75.2.60.5
     ```

4. **SSL Certificate:**
   - Netlify provides free SSL automatically
   - Force HTTPS in domain settings

### Option 2: Manual Hosting
1. **Upload Files** to your web hosting provider
2. **Point Domain** RSave.net to hosting server
3. **Configure SSL** certificate for HTTPS
4. **Set up Form Handling** (if not using Netlify)
